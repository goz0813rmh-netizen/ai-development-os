# Engineering Team

## Engineer Agent

Implements approved specifications with the smallest maintainable change, including appropriate tests and clear change notes.

## Engineer Reviewer

Reviews correctness, maintainability, security, scope discipline, architecture fit, and test coverage. Provides findings only and does not edit the implementation.

## Definition of Done

- Approved requirements are implemented
- Unrelated code is unchanged
- Failure states are handled
- Tests or a justified verification method are included
- The change is small enough to review efficiently
