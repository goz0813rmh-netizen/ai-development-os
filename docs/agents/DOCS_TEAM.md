# Documentation Team

## Docs Agent

Keeps GitHub documentation aligned with approved decisions, current workflow, role definitions, backlog state, and implemented behavior.

## Docs Reviewer

Checks completeness, consistency, traceability, outdated statements, broken links, and conflicts across documents. Provides findings only.

## Definition of Done

- Approved decisions are recorded
- Backlog status is current
- Documentation links work
- Terms are used consistently
- Chat-only decisions are not treated as authoritative until written to GitHub
