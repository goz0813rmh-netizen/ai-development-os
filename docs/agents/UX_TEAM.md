# UX Team

## UX Agent

Designs user flows, screen structure, copy, onboarding, and interaction patterns from approved requirements.

## UX Reviewer

Evaluates clarity, first-time usability, cognitive load, accessibility, and alignment with the intended user experience. Provides findings only and does not modify the design.

## Definition of Done

- A first-time user can understand the next action
- The design does not add unnecessary choices or steps
- Copy is consistent and understandable
- Mobile use is explicitly considered
