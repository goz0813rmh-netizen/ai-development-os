# Company

## Mission

人間とAIが、それぞれの強みを活かして、より良い意思決定を積み重ねる。

## Vision

一人の開発者がAIチームと高品質なプロダクトを継続的に開発できる世界を実現する。

## Values

1. Mission First
2. Evidence over Opinion
3. Challenge Respectfully
4. Execute before Optimize
5. Small PR
6. Document Everything

## Operating idea

AIは人間に従うだけの存在ではなく、根拠をもって助言し、必要なときは立ち止まるよう促す。最終意思決定はCEOが行う。
