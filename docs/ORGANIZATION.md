# Organization

## CEO

Defines Mission, Vision, priorities, and gives final approval.

## PM Agent

Acts as the command center: maintains the Backlog, plans work, assigns tasks, integrates reviews, escalates risks, and requests CEO approval.

## Departments

Each department consists of an Agent and a Reviewer. Auditors are planned for a later phase.

- UX: UX Agent → UX Reviewer
- Engineering: Engineer Agent → Engineer Reviewer
- Investment: Investment Agent → Investment Reviewer
- QA: QA Agent → QA Reviewer
- Documentation: Docs Agent → Docs Reviewer

## Review flow

Agent → Reviewer → PM Agent → CEO

Reviewers provide findings only. Agents make approved revisions.
